### Neler Yaptım?
* **Arayüz Tasarımı:** Figma tasarımına sadık kalarak notifications sayfasını Tailwind CSS ile kodladım.
* **Filtreleme Sistemi:** Kullanıcıların bildirimler arasında kolayca arama yapabilmesi için tarih (oldest/newest) ve kategori bazlı filtreleme seçenekleri ekledim.
* **Backend Bağlantısı:** Localhost:5000 üzerinde çalışan sunucu ile API üzerinden haberleşerek bildirimlerin anlık olarak listelenmesini, eklenmesini ve silinmesini sağladım. (İstenilen CRUD işlemlerini tamamladım.)

### Kurulum
1. Bağımlılıklar : `npm install`
2. Çalıştırmak için : `npm run dev` (Varsayılan: http://localhost:3000)
